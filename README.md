# Setup

Clone this repository.

```bash
$ git clone <this project>
```

## Step 1 : Install dependencies. Make sure you already have `nodejs`, `npm` and `yarn` installed in your system.

```bash
$ yarn
```

## Step 2 : Running locally

```bash
$ npm run start
```
